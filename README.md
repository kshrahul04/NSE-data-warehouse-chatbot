# 🏦 NSE Data Warehouse Chatbot

An NLP-powered chatbot built during my internship at NSE (National Stock Exchange of India) that enables non-technical users to query enterprise data warehouses using plain English.

## 📌 Overview
Non-technical teams at large organizations often struggle to retrieve data from enterprise warehouses due to the complexity of SQL. This chatbot bridges that gap by translating plain English questions into accurate SQL queries automatically.

## 🎯 Features
- Plain English to SQL query translation
- Keyword mapping to database schema
- Support for complex data retrieval without SQL knowledge
- Reduced analyst query turnaround time
- Built for NSE's enterprise data warehouse infrastructure

## 🛠️ Tech Stack
- **Database:** SQL, Cloudera (Hadoop Ecosystem)
- **Tools:** VS Code, Excel
- **Domain:** Data Warehousing, NLP, Enterprise Data Management

## 🔍 How It Works
User types plain English
↓
Chatbot identifies keywords
↓
Keywords mapped to database schema
↓
SQL query generated automatically
↓
Data retrieved from warehouse
↓
Results returned to user

## 📈 Impact
- Enabled non-technical staff to independently query enterprise data
